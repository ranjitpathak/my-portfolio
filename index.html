Why it matters: Fixes information lags between departments, giving floor coordinators a real-time view of batch stages.

Power Pivot DAX Modeling

Dynamic Inventory Optimization Dashboard

An analytical data layout mapping inventory turns and material usage ranges using DAX data tables. It matches actual item consumption patterns against minimum safe buffer targets.

Why it matters: Protects the assembly tracks against materials running out unexpectedly without overloading working warehouse shelves.

PowerShell Regex Logic

Cross-Network Image Search Tool

A custom utility that crawls multiple mapped storage partitions at once. By evaluating file strings with Regular Expressions (Regex), it instantly bundles images into an aggregated catalog view.

Why it matters: Prevents floor supervisors from losing valuable time digging through nested shared network directories to verify batch designs.
